# Screenshots

Add screenshots of the dashboard here:

- `dashboard-preview.png` - Main dashboard overview
- `dive-table.png` - Dive table with detail view
- `gas-analysis.png` - Gas analysis charts
- `trips.png` - Trip summary cards

To capture screenshots:
1. Run the dashboard generator on your dive data
2. Open the HTML file in a browser
3. Take screenshots of each tab
4. Save them in this folder
